水平分割线 ==========================css选择器=================================== .nav-bar 类名是 nav-bar 的所有标签 #usename id 是 username的所有标签 p 所有的p标签 p,h1 找到所有的p标签和h1标签 div p div内部的所有p标签 div>p 父元素是div的每个p标签 div+p div紧邻的每个p标签 p~ul 相同父元素，p标签后面的所有 ul 标签 a[target] 有target属性的所有a标签 a[target=_blank] target属性是_blank的所有a标签 !!! 灵活组合多个选择器。 --------------了解---------------------- a[href^="https"] 选择href属性以https开头所有a标签 a[href$="xxx"] 以 xxx 结尾的所有a标签 a[href*="google"] 包含 google 的所有a标签 -------------------------------------------------- p:first-child 如果p标签是它父元素的一个子标签，那么选中 ----------------------------------------------- [title~=flower] title属性中包含flower的所有标签 [title|=flower] title属性中以flower开头的所有标签 ----------------------------------------------- a:link 所有未被访问的链接 a:visited 选择所有已被访问的链接 a:active 选择活动链接 a:hover 选择鼠标指针位于其上的链接 ----------------------------------------------- input:focus 获得每个焦点(选中)的input元素 -----------------------------------------------
